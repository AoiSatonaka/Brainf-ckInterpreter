# Brainf*ck InterPreter

## How To Use

### shell

``` shell
go get github.com/AoiSatonaka/Brainf-ckInterPreter/pkg/bfi
```

### go

``` go
import "github.com/AoiSatonaka/Brainf-ckInterPreter/pkg/bfi"

source := []rune("++++++++[>++++++++<-]>+.+.+.")
bfi.InterPret(source,nil)
```
